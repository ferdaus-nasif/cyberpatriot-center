Windows Checklist (Under Construction)

DO NOT PROCEED WITHOUT INVESTIGATING THE FORENSICS QUESTIONS AND THE README FILE.

  1. Windows Updates 🛠️
    1. Just checking updates and updating it.
    2. In harder levels they will make it harder to update. They will mess with the registry and disable updates.
  2. Firewall Settings 🔥🧱
    1. Search up Windows Firewall
    2. Make sure it's on.
    3. If any apps need permissions for it then you may enable it.
  3. Local Security Policies 🔒
    1. Type in secpol (aka Local Security Policies)
    2. Open Account Policies → Password Policy.
    3. Change Enforce Password History to 7
    4. Set Maximum Password Age to 99 days
    5. Set Minimum Password Age to 15 days
    6. Set Minimum Password length to 10 Characters
    7. Make sure to enable "Password must meet complexity requirements"
    8. Head to → Account Lockout Policy
    9. Set Account Lockout Threshold to 6
    10. Set Account Lockout Duration to 0
    11. Reset account lockout counter after 15
    12. Go to Local Policies → Audit Policy & set to record every type of audit with both success and failure
  4. Remote Desktop 🖥️
    1. Go to Settings → 🔍 Remote Desktop Settings
    2. If nothing is mentioned Disable “RD”
  5. Other Users 🧑
    1. Change passwords of weak users (There will ALWAYS be one)
    2. Remove Admin Perms from user that was given the wrong role.
  6. Quick Rapid Fire Things 🔥
    1. Run a virus scan
    2. Check for Protection Updates
  7. Virus & Threat Protection 🛡️
    1. In App & Browser Protection change the settings below
    2. Set Check apps and files to warn (if not)
    3. And review other settings that are wrong and need to be fixed.
    4. Make sure in Virus & Threat Protection settings are set to be protected/enabled and working
  8. Install Important Programs 📲
    1. Install the latest stable edition of Notepad++
    2. If Firefox is not installed then install it
