Windows Checklist

DO NOT PROCEED WITHOUT INVESTIGATING THE FORENSICS QUESTIONS AND THE README FILE.

  1. Windows Update
    1. Install all updates.
    2. Restart until fully updated.
    3. Update required third-party software only.
  2. Users and Groups
    1. Compare every account to the README.
    2. Disable Guest and Administrator unless required.
    3. Remove unauthorized administrators.
    4. Do not modify WDAGUtilityAccount or DefaultAccount unless instructed.
  3. Password Policy
    1. History: 24
    2. Maximum age: 30
    3. Minimum age: 5
    4. Minimum length: 14
    5. Complexity enabled
    6. Reversible encryption disabled
  4. Account Lockout
    1. Threshold: 5-10 attempts depending on README.
    2. Reset counter: 15 min.
    3. Duration: 30 min.
  5. Audit Policy
    1. Never blindly enable every audit setting.
    2. Match README exactly.
    3. Advanced Audit Policy overrides basic policy.
  6. Security Options
    1. Review all settings.
    2. Prioritize UAC, NTLMv2, Guest, Administrator, anonymous enumeration, blank passwords.
    3. Avoid forcing FIPS/Smart Card/Kerberos unless README requires.
  7. Firewall
    1. All profiles ON.
    2. Review inbound rules.
    3. Do not blindly block web/server ports.
    4. Remove malicious rules.
  8. Windows Defender
    1. Definitions updated.
    2. Real-time protection on.
    3. Check Tamper Protection.
    4. Review exclusions.
    5. Run Full Scan early.
    6. Review Protection History.
  9. Services
    1. Disable Telnet, Remote Registry, SMB1, TFTP when not required.
    2. Leave IIS, DNS, FTP, SSH, WinRM, SMTP, RDP if image requires them.
    3. Research unknown services before disabling.
  10. Installed Programs
    1. Check README for what programs are permitted.
    2. Remove unauthorized software.
    3. Use Programs and Features first.
    4. Don't uninstall the scoring engine.
  11. Scheduled Tasks
    1. Inspect every folder.
    2. Delete malicious tasks.
    3. Keep required tasks.
  12. Startup Apps
    1. Disable unnecessary startup items.
    2. Remove malicious startup items.
    3. Keep required startup items.
    4. Leave Defender and required security tools enabled.
  13. Hosts File
    1. Remove malicious redirects.
    2. Keep localhost entries.
    3. Preserve README-required entries.
  14. PowerShell
    1. Disable PowerShell v2.
    2. Use RemoteSigned if appropriate.
    3. Enable logging only if required by image.
  15. Optional Features
    1. Disable SMB1.
    2. Disable Telnet Client.
    3. Leave IPv6 unless README says otherwise.
    4. Review Hyper-V/WSL/IIS based on image.
  16. Malware Hunting
    1. Check running processes.
    2. Use netstat -ano.
    3. Review Task Manager.
    4. Search user folders for suspicious executables.
    5. Inspect Defender quarantine.
  17. Quick Commands
    1. lusrmgr.msc
    2. secpol.msc
    3. gpedit.msc
    4. services.msc
    5. wf.msc
    6. taskschd.msc
    7. compmgmt.msc
    8. optionalfeatures
    9. ncpa.cpl
    10. appwiz.cpl
    11. Get-LocalUser
    12. Get-Service
    13. Get-Process
    14. Get-MpComputerStatus
